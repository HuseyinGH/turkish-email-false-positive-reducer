# Notebook'lar

Bu klasöre Kaggle'dan indirdiğin notebook(ları) ekle:

- `01_spam_classifier_training.ipynb` — Spam tespit modeli eğitimi
- `02_mt5_softening_training.ipynb` — Yumuşatma modeli eğitimi + değerlendirme

## Kaggle'dan İndirme

1. Kaggle'da notebook'u aç
2. File → Download Notebook
3. İndirilen .ipynb dosyasını bu klasöre koy
4. Dosya adını yukarıdaki gibi düzenle

Notebook çok büyükse (>50MB), Kaggle'da "Clear all outputs" yapıp
tekrar indir.
